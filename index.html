Privacy Policy

Effective Date: June 13, 2025

This Privacy Policy describes how the "Dm msgs" app uses and protects any information received through Instagram Messenger using Meta’s Graph API.

1. Purpose

This app is developed solely for educational and testing purposes to understand and evaluate the working and capabilities of the Instagram Graph API, particularly for messaging features.

2. Data Collection

The app may collect the following data during testing:

  • Instagram direct messages (DMs)
  • Sender Instagram ID and username
  • Timestamp of messages

All data is accessed strictly through permissions granted via the Meta Developer platform with user consent (via roles or test users).

3. Data Usage

Data collected is used only for development, debugging, and testing API responses. It is not stored, shared, or used for any commercial or advertising purposes.

4. Data Retention

No long-term data storage is implemented. All message data is processed temporarily and discarded after use in the development session.

5. Data Deletion

Test users may request deletion of any session data by contacting the developer at the email below. All data related to such a request will be immediately deleted.

6. Contact
